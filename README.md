## Idea相关配置
### 1.自动生成文件注释
    File -> Settings -> Editor -> File and Code Templates   
```java
/**
* @Author FLY
* @Createdate ${YEAR}-${MONTH}-${DAY} ${TIME}
* @ProjectName ${PROJECT_NAME}
* @Desc ${PACKAGE_NAME}
*/
```
### 2.设置编程文字样式
    File -> Settings -> Editor -> Font
    设置样式为“consolas”，字体大小一般为16
